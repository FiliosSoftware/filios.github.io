IAHE-Mobile

IAHE cross-platform mobile application.

View on GitHub

Private Bug Reporter

Public Bug Reporter

Goals:

  1. Create a mobile app with some general IAHE info that be useful on a phone.
    • IAHE Contact Info
    • Indiana law on homeschooling
    • News
      • Notifications when there is a legislative update, a new issue of the Informer magazine, convention announcements, etc.
    • Convention Schedule
    • Capitol Day Information
  2. Offline capability.
  3. Available on these platforms:
    • Android
    • iOS
    • Windows

Tools:

  1. GitHub
  2. Jira
  3. Jenkins
  4. Visual Studio 2015 and Apache Cordova Tools for VS
  5. Apache Cordova
  6. Ionic Framework
